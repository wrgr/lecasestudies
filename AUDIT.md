# Citation Audit — Capability Matters

This document records the citation audit performed on the casebook,
distinguishes between quotes that have been verified, quotes that have
been reworded as paraphrases, and the residual items that still need
human review before any printed edition.

The audit covered 97 case quotes, 22 numbered citations in the
introduction, and roughly 300 bibliographic items in case sources,
literature lists, and the References section.

---

## Method

For each quote attributed to a specific source, an independent web check
was performed (across two batched verification passes) and the result
was classified as:

- **VERBATIM** — the quote matches the source's wording
- **CLOSE** — the substance and most words match; minor wording differs
- **PARAPHRASE** — the substance is correct, the wording is the author's
- **LIKELY FABRICATED** — no evidence the quote exists in this form

Anything that did not return a **VERBATIM** result was rewritten so the
attribution begins with "Paraphrasing…" or "Author summary of…", which
is an honest and defensible representation of the underlying source.

The same approach was applied to citation entries in source lists and
further-reading items: bibliographic references where the work was
confirmed real are left in place; references where author/year/title
details were wrong are corrected; references that could not be confirmed
have been replaced with verified alternates.

## Flagged for manual review

### Substantive (factual) concerns

**Case 38: GIFT Discontinuation.** The casebook treats the
Generalized Intelligent Framework for Tutoring as effectively
discontinued, drawing on the project context provided at the start of
the project. Independent web verification suggests GIFT may still be
active as an ARL research prototype (gifttutoring.org). Two
possibilities:

1. The "discontinuation" framing reflects a 2023 wind-down that the
   project context describes accurately but that has not been widely
   reported.
2. The framing is wrong and Case 38 should be reframed — e.g. as
   "GIFT Adoption Gap" — describing the institutional adoption
   shortfall rather than discontinuation.

Recommend: verify the program's current status with ARL or with the
LDT/LENS program leadership before any printed edition. The case's
substantive point — that GIFT has not achieved the institutional
adoption its evidence base warrants — is well-supported either way.

**Case 91: Singapore Airlines — Phoenix Award citation.** The
casebook had quoted Air Transport World's 2006 Phoenix Award citation;
verification could not confirm the exact wording. The quote has been
reworded as author framing. The substantive claim (Singapore Airlines
as a sustained capability-investment exemplar) is well-supported.

## Recommended next steps before any printed edition

1. **Substantive review** — confirm Case 38 (GIFT) status with ARL
   and reframe if necessary.
2. **Single-source verification pass** — for each case, verify the
   primary report citation against the issuing institution's website
   (NTSB, GAO, CSB, etc.). This is a ~4-hour exercise.
3. **Quote reverification** — pull the actual report text for the
   ten or so quotes that are still presented as verbatim in the case
   bodies (not just in the pull-quote block) and confirm the wording.
4. **Add DOIs / URLs** to References — every reference in
   `backmatter/references.typ` should carry a DOI or stable URL.
   Approximately one third currently do.
5. **Index of cases** — for the printed edition, an alphabetical
   index of case names + first-mention page would help cross-referencing
   between cases (many cases reference others as "Case X").
